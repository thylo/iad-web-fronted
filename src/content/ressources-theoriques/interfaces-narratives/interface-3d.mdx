## Intérêt narratif

- **Sensation de liberté maximale**
- **Narration environnementale** (l'espace raconte)
- **Vision moins dirigiste** de la narration
- Présence et immersion profondes

## Enjeux et risques

- **Difficile de guider sans diriger**
- **Complexité technique énorme** (3D, performances, VR)
- **Comment raconter si l'utilisateur ne va pas où prévu?**
- Risque d'ennui (trop de liberté = perte de direction)

## Quand utiliser 3D ?

- ✅ Vous racontez une histoire exploratoire
- ✅ L'univers spatial est central
- ✅ Vous avez ressources technique + artistique
- ✅ VR/immersion enrichit votre propos

## Design

### Guidage subtil

Comment diriger sans forcer :
- Éclairage attire vers zones importantes
- Architecture oriente naturellement
- NPCs placés stratégiquement
- Micro-indicateurs (son, particules)

### Landmarks visuels

Créer repères mémorables :
- Monuments distinctifs
- Variations de terrain/architecture
- Indices visuels pour orientation
- Cohérence spatiale (pas de téléportation)

### Narration environnementale

Raconter via l'espace :
- Design d'intérieur parle du personnage
- Dégradation montre l'histoire
- Détails trouvables révèlent le backstory
- Monde cohérent avec sa propre logique

### Performance critique

- Frame rate stable (surtout VR)
- Pas de lag = immersion brisée
- Optimisation géométrique
- Streaming du contenu

## Variantes

### Monde ouvert
Complète liberté d'exploration.
- Avantage : Sens liberté max
- Inconvénient : Risque de perdre le fil

### Monde semi-ouvert
Zones explorables librement, limites
- Avantage : Direction + liberté
- Inconvénient : Peut sembler artificiel

### Couloir linéaire 3D
Chemin linéaire prédessiné en 3D.
- Avantage : Contrôle narratif
- Inconvénient : Pas vraie liberté

### Réalité virtuelle (VR)
Véritable immersion 360°.
- Avantage : Empathie/présence max
- Inconvénient : Sickness VR, coût

## Combinaisons avec autres interfaces

**3D + Point-and-Click**
- Navigation 3D libre
- Interactions via clic sur objets
- Dialogue sur NPC

**3D + Gestural**
- Navigation par gestes naturels
- Interaction tactile avec l'espace

## Bonnes pratiques

1. **Orientation claire** : On sait toujours où on est
2. **Guidage subtil** : Diriger sans forcer
3. **Performance** : Jamais de lag
4. **Espace significatif** : Chaque zone raconte
5. **Découvrabilité** : Contenu caché mais trouvable

## Risques courants

❌ **Monde vide** : Beau mais sans narration
❌ **Perte d'orientation** : Désorienté après 5 min
❌ **Performance lag** : Immersion cassée
❌ **Trop linéaire** : Faux liberté
❌ **Complexité technique** : Sur-ambitieux

## Solutions

✅ Narration environnementale (pas juste beau)
✅ Landmarks, compass, mini-map si besoin
✅ Optimisation agressif dès le start
✅ Zones exploratoires + hub structure
✅ Prototypage intensif pour scope
✅ User testing pour orientation/guidage

## Considérations VR spéciales

- **Sickness VR** : Mouvement fluide, pas de téléportation brutale
- **Confort** : Pas d'escalade impossible
- **Durée** : VR fatigue après 15-20min
- **Accessibilité** : Options pour handicaps
