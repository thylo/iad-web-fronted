## Intérêt narratif

- **Décolle l'interaction d'une décision rationnelle**
- **Crée connexion émotionnelle par le geste**
- **Approche expérimentale/artistique**
- Corporalité de l'interaction

## Enjeux et risques

- **Trouver le geste qui sert vraiment l'émotion**
- **Estimer le niveau de dextérité requis**
- **Risque de gadget** si le geste n'est pas signifiant
- Accessibilité pour handicaps moteurs

## Quand utiliser gestural ?

- ✅ Vous racontez une histoire émotionnelle/viscérale
- ✅ Le geste enrichit le propos narratif
- ✅ Public sur mobile/tactile
- ✅ Vous visez expérience artistique

## Design

### Geste signifiant

Le geste doit être **motivé** :
- Swipe pour tourner page (naturel)
- Dessiner pour créer (poétique)
- Secouer pour déclencher (émotion)
- Pas de gestes arbitraires

### Accessibilité gestuelle

Pas tout le monde peut faire tous les gestes :
- Fournir alternatives (clics, touches)
- Documenter clairement les gestes
- Tester avec utilisateurs diversos
- Respecter limites physiques

### Feedback tactile

- Vibration confirme le geste
- Animation répond immédiatement
- Son renforce l'interaction
- Sensorialité multimodale

### Progression

- Commencer simple (swipe)
- Complexité augmente graduellement
- Pas de gestes impossibles
- Tester avec gamers/non-gamers

## Gestes courants

### Swipe
Balayer dans une direction.
- **Utilité** : Tourner pages, naviguer
- **Intuitif** : Très

### Pinch / Spread
Pincer ou écarter deux doigts.
- **Utilité** : Zoom, ouverture
- **Intuitif** : Moyen (mobile-only)

### Draw / Sketch
Dessiner à l'écran.
- **Utilité** : Création, interaction créative
- **Intuitif** : Faible (nécessite pratique)

### Tap
Taper/frapper l'écran.
- **Utilité** : Interrupteur, action rapide
- **Intuitif** : Très

### Shake
Secouer l'appareil.
- **Utilité** : Déclencher événement, réaction
- **Intuitif** : Moyen (amusant)

### Tilt / Rotate
Incliner ou tourner l'appareil.
- **Utilité** : Navigation, contrôle mouvement
- **Intuitif** : Moyen (sur mobile)

## Variantes

### Single geste
Seul type de geste utilisé.
- Avantage : Simplicité, focus
- Inconvénient : Limité narrativement

### Multi-gestes
Plusieurs gestes pour actions différentes.
- Avantage : Richesse d'interaction
- Inconvénient : Courbe d'apprentissage

### Gestes contextuels
Geste change sens selon contexte.
- Avantage : Flexibilité
- Inconvénient : Confusion possible

## Combinaisons avec autres interfaces

**Gestural + 3D**
- Navigation 3D par gestes naturels
- Interaction immersive

**Gestural + Scrollable**
- Gestes + scroll dans même expérience
- Interaction multimodale

## Bonnes pratiques

1. **Geste motivé** : Pourquoi ce geste?
2. **Apprentissage graduel** : Tutoriel intégré
3. **Feedback immédiat** : Confirmation du geste
4. **Alternative accessible** : Pas seulement gestes
5. **Cohérence** : Même geste = même action

## Risques courants

❌ **Geste arbitraire** : Pas lié au sens
❌ **Trop de gestes** : Impossible à mémoriser
❌ **Pas d'alternatives** : Inaccessible
❌ **Feedback flou** : On ne sait pas si on a réussi
❌ **Dextérité excessive** : Inaccessible

## Solutions

✅ Limiter à 2-3 gestes clés
✅ Geste lié thématiquement au sens
✅ Tutoriel dans la narration
✅ Feedback haptique + visuel
✅ Alternative pour handicaps moteurs
✅ Tester sur vrais utilisateurs

## Considérations d'accessibilité

- **Handicaps moteurs** : Gestes impossibles
- **Vision** : Feedback audio/haptique
- **Cognitif** : Trop de types de gestes
- **Âge** : Dextérité varie avec l'âge
