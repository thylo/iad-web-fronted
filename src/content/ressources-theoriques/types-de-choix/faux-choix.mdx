## Qu'est-ce qu'un faux choix ?

**Exemple typique :**

```
Le jeu : "Voulez-vous sauver Marie ou Jean ?"
Vous : [Choisissez Jean]
Le jeu : "Vous avez sauvé Jean !"
[5 minutes plus tard]
Le jeu : "Marie est sauvée aussi par quelqu'un d'autre"
```

**Le contrat rompu :**
- L'interface disait : "Votre choix aura des conséquences"
- La réalité : "Votre choix n'a changé rien"

## Pourquoi c'est un problème

### 1. Rupture du contrat

Le public accepte l'interactive narrative basée sur un contrat implicite : "Mes choix importent."

Les faux choix rompent ce contrat.

### 2. Sentiment de manipulation

"L'auteur m'a trompé. Je ne lui fais plus confiance."

Après un faux choix, le public doute de tous les choix suivants.

### 3. Perte d'immersion

"Si ce choix ne change rien, pourquoi devrais-je continuer ?"

L'agentivité disparaît, l'engagement s'effondre.

### 4. Frustration viscérale

C'est différent de l'exploratoire honnête.
- Exploratoire honnête : "C'est un détail"
- Faux choix : "C'était censé compter"

## Types de faux choix

### Le cosmétique

Seule la phrasing change, pas les événements.

**Exemple :**
- "Allez par la porte ou la fenêtre"
- Les deux mènent au même endroit
- Seul le dialogue change

**Problème :** Sentiment que c'était du temps gaspillé

### Le convergence tardive

Les choix divergent puis convergent brusquement.

**Exemple :**
- Choisir A vs B
- 10 minutes de contenu différent
- Puis : "Mais les deux personnages se retrouvent de toute manière"

**Problème :** L'investissement émotionnel est soudainement annulé

### Le faux dilemme

Les deux options menacent au même endroit.

**Exemple :**
- "Sauvez la ville ou la forêt"
- Vous choisissez la forêt
- Vous découvrez que la ville était inévitable

**Problème :** Vous avez fait un choix moral sans vraie base

### Le choix révélé comme illusion

L'univers narr admet que le choix n'existait pas.

**Exemple :**
- "Vous aviez le choix"
- [Révélation] "Mais vous n'aviez vraiment pas le choix"
- Peut être intentionnel (meta) mais frustrant

**Problème :** Dépend si c'est ironique ou vraiment faux

## Faux choix intentionnels

Parfois, un faux choix est **voulu**.

### Rupture du 4e mur

**Exemple : The Stanley Parable**
- "Vous avez le choix"
- Toutes les options mènent au même endroit
- C'est une critique de l'illusion du choix

**Quand c'est efficace :** Public comprend que c'est méta-critique

**Quand ça échoue :** Public pense juste que c'est mauvais design

### Simulation d'agentivité

**Exemple : Narrative game where you "choose" but can't really**

Intentionnel si le message est clair : "Vous n'aviez jamais le choix"

Non-intentionnel si c'est juste mal fait.

## Comment identifier un faux choix

### Test de conséquence

- Rejouez avec un choix différent
- Voyez-vous vraiment du contenu différent?
- Ou est-ce exactement le même?

### Test d'impact

- Après le choix, y a-t-il un changement visible?
- Pas de changement = probablement faux

### Test de durabilité

- L'effet du choix dure-t-il?
- Ou est-il immédiatement annulé?

### Test de rejoueabilité

- Avez-vous envie de rejouer pour voir l'autre choix?
- Si non, c'était probablement faux

## Comment ÉVITER les faux choix

### 1. Soyez clair sur la nature du choix

**Si exploratoire :**
- "Découvrez l'une ou l'autre histoire"
- C'est honnête

**Si rationnel :**
- "Ce choix affectera la fin"
- Et VRAIMENT affecter la fin

### 2. Convergence honnête

Si les chemins convergent, le signaler:
- "Tous les chemins menent à Rome"
- C'est acceptable si c'est honnête

### 3. Hiérarchiser les choix

- Certains choix sont mineurs
- D'autres sont majeurs
- Classer clairement

### 4. Tester chaque chemin

- Rejouez avec chaque choix
- Voyez-vous vraiment des différences?
- Pas de raccourcis: TESTER

## Exemple de bon design avec convergence

```
                    Objectif final
                    /    /    \    \
                  /    /        \    \
            Chemin A  Chemin B  Chemin C  Chemin D
            |         |        |        |
        Choix 1a    Choix 1b   Choix 1c  Choix 1d
```

- Chaque choix mène à un chemin différent
- Tous les chemins convergent à un point commun
- MAIS : le voyage était différent
- MAIS : la transition est narrativement justifiée

**Exemple réel : Life is Strange**
- Choix majeurs affectent le voyage
- Tous convergent vers un climax commun
- Mais comment on y arrive change vraiment

## Quand les faux choix peuvent marcher

### Si c'est métanarratif

- Le jeu reconnaît que c'étaient des faux choix
- C'est une critique intentionnelle
- **BUT:** Le public doit **comprendre** que c'est intentionnel

### Si c'est exploratoire honnête

- "Voici plusieurs perspectives"
- Elles coexistent sans hiérarchie
- C'est clairement exploratoire

## Distinguer faux choix vs choix exploratoire

| Aspect | Choix exploratoire | Faux choix |
|--------|-------------------|-----------|
| **Contrat** | "Découvrez du bonus" | "Ce choix compte" |
| **Conséquence** | Aucune prévue | Prétend en avoir |
| **Intégrité** | Honnête | Trompeuse |
| **Sentiment** | Enrichissement | Frustration |

## Leçons à retenir

✅ **FAIRE** : Choix rationnel honnête avec vraies conséquences
✅ **FAIRE** : Choix exploratoire honnête sans prétendre à plus
✅ **FAIRE** : Méta-critique intentionnelle sur les faux choix
❌ **NE PAS FAIRE** : Faux choix accidentels qui trompent

## Analyse critique

Pour chaque expérience interactive vous analysez :
- Y a-t-il des faux choix?
- Sont-ils intentionnels?
- Comment le public réagit-il?
- L'intégrité narrative est-elle sauvée?

## Pour votre propre création

**Questions fondamentales :**
1. Ce choix a-t-il vraies conséquences?
2. Ai-je les ressources pour les produire?
3. Si non, ai-je le courage de le dire explicitement?
4. Si c'est une critique meta, est-ce clair?

**Règle d'or :** Respectez le contrat avec votre public.
