## Ce qui fonctionne

Les joueurs enquêtent physiquement (cartes, indices) mais accèdent à une vraie base de données web (Antares Database) pour vérifier des informations, consulter des dossiers, interroger des témoins virtuels.

## Inspiration pour vos projets

Mélange réel et virtuel pour créer une enquête qui déborde du plateau. Le web devient un outil d'investigation comme dans la vraie vie.
