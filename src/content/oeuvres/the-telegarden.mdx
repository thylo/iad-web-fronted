## Ce qui fonctionne

Les visiteurs du site web pouvaient contrôler un bras robotique réel pour planter et arroser des graines dans un vrai jardin. Œuvre pionnière d'interaction web/monde physique.

## Inspiration pour vos projets

Projet visionnaire datant de 1995 qui démontrait déjà que le web pouvait contrôler le monde physique. Ancêtre des objets connectés. Installation artistique web historique.
