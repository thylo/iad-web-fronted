## Interaction

Switcher entre les fils narratifs. Le spectateur construit son propre ordre de lecture et découvre progressivement les connexions.

## Schéma

```
Récit A: Début → Scène 1 → Scène 2 → Fin A
Récit B: Début → Scène 1 → Scène 2 → Fin B
Récit C: Début → Scène 1 → Scène 2 → Fin C
         ↕         ↕         ↕
    [Le spectateur peut basculer]
```

## Avantages

- Richesse narrative considérable
- Le spectateur construit sa propre compréhension
- Révèle les connexions et parallèles
- Rejouabilité (ordre de découverte différent)

## Inconvénients

- Difficile de suivre plusieurs fils simultanément
- Peut être frustrant (FOMO - peur de rater quelque chose)
- Production très lourde (plusieurs récits complets)
- Risque de déséquilibre entre les histoires

## Quand l'utiliser

- Récits chorale (plusieurs personnages équivalents)
- Exploration d'une communauté/ville par plusieurs habitants
- Quand les connexions entre histoires sont le propos central
- Documenter un événement vu par plusieurs témoins
- Montrer des causales croisées

## Variantes

- **Parallèle convergente** : Les récits se rejoignent vers une fin commune
- **Parallèle divergente** : Les récits s'éloignent progressivement
- **Parallèle avec embranchements** : Combinaison avec des choix dans chaque fil
