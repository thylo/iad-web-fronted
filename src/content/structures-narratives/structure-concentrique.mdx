## Interaction

Choisir quelle perspective, époque ou couche explorer. Pas d'ordre imposé, navigation libre entre les couches.

## Schéma

```
      Perspective A
           ↓
Perspective D ← [ÉVÉNEMENT] → Perspective B
           ↑
      Perspective C
```

## Avantages

- Révèle la complexité d'une situation
- Encourage la réflexion critique
- Rejouabilité naturelle (explorer chaque perspective)
- Pertinent pour les événements controversés

## Inconvénients

- Peut être répétitif si mal exécuté
- Difficile de maintenir l'intérêt sur toutes les perspectives
- Risque de confusion chronologique
- Impression de fragmentation narrative

## Quand l'utiliser

- Histoires avec plusieurs protagonistes
- Événements controversés nécessitant plusieurs points de vue
- Exploration spatiale d'un lieu à travers le temps
- Documentaires sur des faits complexes
- Œuvres pédagogiques nécessitant la compréhension multidimensionnelle

## Variantes

- **Temporelles** : Un lieu exploré à différentes époque (passé, présent, futur)
- **Perspectivistes** : Un événement vu par différents personnages
- **Thématiques** : Un sujet exploré selon différents angles (émotionnel, factuel, artistique)
