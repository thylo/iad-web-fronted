## Interaction

Explorer librement. Chaque élément narratif mène à d'autres selon une logique de connexions, pas de chemin prédestiné.

## Schéma

```
   A ─── B
   │  ╲  │ ╲
   │   ╲ │  ╲
   C ─── D ─── E
    ╲   ╱ │   ╱
     ╲ ╱  │  ╱
      F ──┴─ G
```

## Avantages

- Liberté totale d'exploration
- Chacun construit son propre récit
- Rejouabilité infinie
- Pertinent pour l'exploration poétique ou conceptuelle

## Inconvénients

- Peut être désorientant
- Difficile de garantir une expérience satisfaisante
- Pas adapté aux récits avec climax/résolution
- Production très complexe et risquée

## Quand l'utiliser

- Exploration poétique
- Bases de données (archives, mémoires)
- Quand le voyage importe plus que la destination
- Œuvres expérimentales
- Installations web narratives

## Variantes

- **Constellation guidée** : Des indices subtils sans imposer un chemin
- **Constellation avec attracteurs** : Certains nœuds sont plus "visibles" que d'autres
- **Constellation temporelle** : Les connexions changent selon le temps de visite
