Dans cette partie, vous allez apprendre à donner du style à votre site d'histoires avec la typographie et les couleurs.

### Ce que vous allez apprendre

Dans ce cours, nous allons couvrir :

1. **Typographie web : les bases** - Les propriétés CSS essentielles
2. **Google Fonts** - Intégrer des polices personnalisées
3. **Polices locales** - Utiliser @font-face
4. **Couleurs CSS** - Hex, RGB, HSL et variables CSS
5. **Palettes de couleurs** - Créer une palette cohérente
6. **Dégradés et backgrounds** - Effets visuels avancés
7. **Exercices pratiques** - Mise en application complète
8. **Checklist et ressources** - Valider vos connaissances

## Fil rouge : Votre système de design

Tout au long de ce cours, vous allez créer un système de design complet pour votre site d'histoires :

### Livrables attendus

À la fin de ce cours, vous devrez avoir :

1. **1-2 polices Google Fonts** intégrées et utilisées
2. **Un système de variables CSS** complet (`:root`)
3. **Une palette de couleurs** cohérente (3-5 couleurs)
4. **Une échelle typographique** définie (h1-h6)
5. **Des styles appliqués** sur toutes vos pages
6. **Une identité visuelle** reconnaissable

### Adaptation par public cible

- **6-8 ans** : Polices rondes, couleurs vives, grandes tailles
- **8-12 ans** : Polices lisibles, couleurs équilibrées, standard
- **Ados** : Graphisme sophistiqué, palette sobre, moderne
