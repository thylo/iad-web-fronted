## À propos de ce journal

Ce journal documente les évolutions majeures du site. Il suit le format inspiré de [Keep a Changelog](https://keepachangelog.com/) :

- **Ajouté** - Nouvelles fonctionnalités ou contenus
- **Modifié** - Changements dans des fonctionnalités existantes
- **Corrigé** - Corrections de bugs ou d'erreurs
- **Supprimé** - Fonctionnalités ou contenus retirés
- **Déprécié** - Fonctionnalités bientôt retirées

## Notes de version

### Philosophie de versioning

Ce site suit une approche de **versioning sémantique adapté** :

- **Version majeure (X.0.0)** : Ajout d'un cours complet ou refonte majeure
- **Version mineure (1.X.0)** : Nouveau module ou section significative
- **Version patch (1.1.X)** : Corrections, améliorations mineures

### Format des entrées

Pour maintenir la cohérence, chaque entrée suit ce format :
- **Date** au format ISO (YYYY-MM-DD)
- **Type de changement** : Ajouté/Modifié/Corrigé/Supprimé
- **Section concernée** : Cours/Ressources/Technique
- **Description** : claire et concise de ce qui a changé
