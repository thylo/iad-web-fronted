## Vue d'ensemble

**Durée :** 2 à 2.5 heures  
**Objectif :** Ouvrir les possibles, découvrir les territoires créatifs, donner l'eau à la bouche

## Timing de la séance

- **30 min** : Ouverture provocante
- **90 min** : Les 5 territoires (exploration + tests + analyse)
- **30 min** : Synthèse et mission d'exploration

---

## Partie 1 : Ouverture provocante (30 min)

### Message d'introduction

> "Le web n'est pas juste pour des sites corporate. C'est un espace de création unique qui combine texte, image, son, interactivité, temporalité. Vous allez explorer ce potentiel et créer des expériences que personne n'a jamais vues."

### Remise en question

**Ce que ce cours N'EST PAS :**
- Un cours de syntaxe JavaScript ligne par ligne
- Un cours où je dicte du code au tableau
- Un cours où l'IA est interdite
- Un cours centré uniquement sur la technique

**Ce que ce cours EST :**
- Un laboratoire créatif avec contraintes techniques
- Un accompagnement pour réaliser vos visions créatives
- Une exploration du web comme medium
- Un cours où l'IA est un outil parmi d'autres

### Votre bagage créatif

Vous maîtrisez déjà :
- **VFX et compositing** (Nuke, After Effects) → vous comprenez le flux d'images
- **3D** (Blender, Maya, Houdini) → vous comprenez la génération procédurale
- **Motion design** → vous comprenez le timing et l'animation

Le web est simplement un nouveau medium avec ses propres possibilités.

### Questions ouvertes (discussion)

- Qu'est-ce qu'un site web pour vous ?
- Quelles expériences web vous ont marqué ?
- Qu'est-ce que le web peut faire que le cinéma/la 3D ne peuvent pas ?

## Partie 2 : Les 5 territoires (90 min)

### Format

Pour chaque territoire (environ 18 min) :
1. **Présentation** (5 min) : concept et question centrale
2. **Exemples concrets** (10 min) : tests d'œuvres existantes
3. **Analyse** (3 min) : formes d'interactivité, rôle du public, structures narratives

---

### Territoire 1 : Expériences muséales/spatiales

**[→ Voir la page complète du territoire](/rtmf1m/territoires/01-museales)**

**Question centrale :** Comment le numérique enrichit le lieu ?

#### Exemples inspirants

**The Boat** (2015) - Documentaire immersif sur un voyage de réfugiés  
- Scroll narratif avec son spatial et animations
- Le médium web permet de contrôler le rythme de découverte
- [Voir dans la galerie](/ressources/galerie/museales#the-boat)

**One Shared House** (2013) - Exploration d'une maison à travers les époques  
- Navigation spatiale dans les pièces et les souvenirs
- Plusieurs familles, plusieurs temporalités
- [Voir dans la galerie](/ressources/galerie/museales#one-shared-house-voir-aussi-dans-lecture)

**Carnets de Versailles** - Application de visite augmentée  
- Contenus contextuels géolocalisés
- Carnet personnel de visite
- [Voir dans la galerie](/ressources/galerie/museales#carnets-de-versailles)

**Questions pour analyser :**
- **Forme d'interactivité :** Machine (exploration d'un système) ou Contributive (si carnet personnel)
- **Rôle du public :** Fantôme (observateur invisible) ou Lui-même (si données personnelles)
- **Structure narrative :** Concentrique (autour d'un lieu) ou Constellation (navigation libre)

[→ Plus d'exemples dans la galerie](/ressources/galerie/museales)

---

### Territoire 2 : Jeux de société hybrides

**[→ Voir la page complète du territoire](/rtmf1m/territoires/02-jeux-societe)**

**Question centrale :** Comment le numérique sert l'expérience collective sans la dominer ?

#### Exemples inspirants

**Chronicles of Crime** (2018) - Jeu d'enquête avec app  
- Scan de cartes physiques pour interroger témoins
- Reconstitution 3D des scènes de crime en VR
- L'app gère la logique, le groupe mène l'enquête
- [Voir dans la galerie](/ressources/galerie/jeux-societe#chronicles-of-crime)

**One Night Ultimate Werewolf** (2014) - Loup-garou accéléré  
- App qui guide la nuit (narration audio)
- Tous jouent simultanément le jour
- Le numérique permet des rôles complexes
- [Voir dans la galerie](/ressources/galerie/jeux-societe#one-night-ultimate-werewolf)

**Mansions of Madness** (2016) - Horreur lovecraftienne coopérative  
- Plateau physique + app pour scénarios procéduraux
- Énigmes numériques intégrées
- [Voir dans la galerie](/ressources/galerie/jeux-societe#mansions-of-madness)

**Questions pour analyser :**
- **Forme d'interactivité :** Machine (système de jeu) + Sociale (interaction entre joueurs)
- **Rôle du public :** Personnage (vous jouez un rôle) ou Divinité (vous manipulez le jeu)
- **Structure narrative :** Embranchements (choix qui changent l'histoire)

[→ Plus d'exemples dans la galerie](/ressources/galerie/jeux-societe)

### Territoire 3 : Expériences de lecture augmentées

**[→ Voir la page complète du territoire](/rtmf1m/territoires/03-lecture)**

**Question centrale :** Comment le medium porte le récit ?

#### Exemples inspirants

**Device 6** (2013) - Roman typographique interactif  
- Le texte dessine le chemin du personnage
- Sons et musique déclenchés par la lecture
- Énigmes intégrées à la narration
- [Voir dans la galerie](/ressources/galerie/lecture#device-6)

**80 Days** (2014) - Tour du monde interactif  
- Adaptation de Jules Verne avec milliers de choix
- Économie, relations, découvertes géographiques
- Chaque partie est unique
- [Voir dans la galerie](/ressources/galerie/lecture#80-days)

**Bandersnatch** (2018) - Film Netflix interactif  
- Choix qui changent radicalement l'histoire
- Méta-narration (le personnage prend conscience des choix)
- 5 fins principales, nombreuses variations
- [Voir dans la galerie](/ressources/galerie/lecture#bandersnatch-black-mirror)

**Phallaina** (2016) - BD scrollable horizontale  
- Première BD pour smartphone pensée pour le scroll
- Animations et profondeur de champ
- Le rythme de lecture devient une interaction
- [Voir dans la galerie](/ressources/galerie/lecture#phallaina)

**Questions pour analyser :**
- **Forme d'interactivité :** Machine (univers narratif fermé)
- **Rôle du public :** Variable selon l'œuvre (Fantôme, Personnage, ou Divinité)
- **Structure narrative :** Embranchements, Linéaire élastique, ou Parallèle

[→ Plus d'exemples dans la galerie](/ressources/galerie/lecture)

### Territoire 4 : Interfaces expérimentales/ludiques

**[→ Voir la page complète du territoire](/rtmf1m/territoires/04-interfaces)**

**Question centrale :** Comment le geste devient l'expérience ?

#### Exemples inspirants

**Patatap** (2014) - Instrument visuel et sonore  
- Chaque touche du clavier = son + animation
- Pas de but, juste l'exploration du geste
- [Voir dans la galerie](/ressources/galerie/interfaces#patatap)

**The Johnny Cash Project** (2010) - Clip collaboratif  
- Dessinez une frame du clip de Johnny Cash
- Des milliers de personnes ont contribué
- Le résultat est une œuvre collective
- [Voir dans la galerie](/ressources/galerie/interfaces#the-johnny-cash-project)

**neal.fun** - Collection d'expériences ludiques  
- "The Size of Space" : explorer l'échelle de l'univers
- "The Deep Sea" : descendre dans les abysses
- "Paper" : jeu d'écriture absurde
- [Voir dans la galerie](/ressources/galerie/interfaces#nealfun)

**Swap Tales** (2016) - Récit manipulable  
- Déplacez physiquement les éléments de l'histoire
- L'interaction tactile change le récit
- [Voir dans la galerie](/ressources/galerie/interfaces#swap-tales)

**Questions pour analyser :**
- **Forme d'interactivité :** Machine ou Contributive (si création collective)
- **Rôle du public :** Divinité (manipulation omnisciente) ou Lui-même (expression personnelle)
- **Structure narrative :** Constellation (exploration libre) ou Linéaire (si parcours défini)

[→ Plus d'exemples dans la galerie](/ressources/galerie/interfaces)

### Territoire 5 : Outils créatifs et générateurs

**[→ Voir la page complète du territoire](/rtmf1m/territoires/05-outils-creatifs)**

**Question centrale :** Comment l'outil devient créatif/ludique/poétique ?

#### Exemples inspirants

**Incredibox** (2009) - Créateur de beatbox  
- Glissez des personnages pour composer de la musique
- Interface intuitive, résultats professionnels
- Ludique ET fonctionnel
- [Voir dans la galerie](/ressources/galerie/outils-creatifs#incredibox)

**Silk** (2010) - Générateur de motifs symétriques  
- Dessinez, l'outil crée des symétries hypnotiques
- Outil simple, résultats complexes
- [Voir dans la galerie](/ressources/galerie/outils-creatifs#silk---interactive-generative-art)

**Pixel Art to CSS** - Convertisseur ludique  
- Dessinez en pixel art, obtenez le code CSS
- Outil technique devenu créatif
- [Voir dans la galerie](/ressources/galerie/outils-creatifs#pixel-art-to-css)

**Looom** (2020) - Animation frame par frame  
- Outil d'animation minimaliste mais puissant
- Contraintes créatives (boucles, couleurs limitées)
- [Voir dans la galerie](/ressources/galerie/outils-creatifs#looom)

**Note pour votre profil RTMF :** Ce territoire peut inclure des outils pour vos workflows VFX/3D, mais ce n'est qu'une option parmi d'autres.

**Questions pour analyser :**
- **Forme d'interactivité :** Machine (outil fermé) ou Contributive (si partage des créations)
- **Rôle du public :** Lui-même (création personnelle)
- **Structure narrative :** Constellation (exploration des possibilités de l'outil)

[→ Plus d'exemples dans la galerie](/ressources/galerie/outils-creatifs)

## Partie 3 : Synthèse et mission d'exploration (30 min)

### Synthèse des territoires

**Questions à poser aux étudiants :**
- Quel territoire vous parle le plus ?

## Mission d'exploration pour la semaine

### 1. Explorer les territoires (obligatoire)

Choisissez **2 territoires** qui vous intéressent et trouvez **3 exemples concrets** pour chacun.

**Pour chaque exemple, notez dans votre carnet :**
- Nom et lien/référence
- Description rapide de l'expérience
- Quel territoire ça représente
- Ce qui fonctionne / ne fonctionne pas
- Forme d'interactivité, rôle du public, structure narrative
- Une capture d'écran ou photo

**Où chercher ?**
- [Galerie complète](/ressources/galerie/) (85+ œuvres)
- [Awwwards](https://www.awwwards.com/) - sites créatifs primés
- [Codrops](https://tympanus.net/codrops/) - expérimentations web
- [neal.fun](https://neal.fun/) - expériences ludiques
- [Incredibox](https://www.incredibox.com/) - outil créatif musical
- [Itch.io](https://itch.io/) - jeux indépendants narratifs

### 2. Commencer le carnet de recherche

Format libre (Notion, Google Docs, carnet papier scanné...) :
- Œuvres explorées (avec analyse)
- Réflexions personnelles
- Idées qui émergent
- Questions techniques
- Inspirations visuelles
