### Le web n'est pas juste pour des sites corporate

C'est un espace de création qui combine texte, image, son, interactivité et temporalité. Vous allez explorer ce potentiel et créer des expériences originales.

### Ce que ce cours N'EST PAS

- ❌ Un cours magistral de syntaxe JavaScript/Svelte ligne par ligne
- ❌ Un cours où l'enseignant dicte du code au tableau
- ❌ Un cours où l'IA est interdite ou ignorée
- ❌ Un cours centré uniquement sur la technique

### Ce que ce cours EST

- ✅ Un laboratoire créatif avec contraintes techniques
- ✅ Un espace d'exploration du web comme medium de création
- ✅ Un accompagnement pour réaliser des visions créatives personnelles
- ✅ Une approche où l'IA est un outil parmi d'autres
- ✅ Un cours centré sur l'expérience, la narration et l'interaction

## Comment ça fonctionne

Vous connaissez VFX, 3D et motion design. Le web est un medium différent avec ses propres règles. Vous utiliserez les outils disponibles (recettes de code, IA, documentation) pour réaliser vos projets.

## Votre projet créatif

Vous créerez un projet personnel dans l'un des **5 territoires créatifs** : expériences muséales, jeux hybrides, lecture augmentée, interfaces expérimentales, ou outils génératifs.

**[→ Découvrir les 5 territoires](/rtmf1m/territoires/)**

## Philosophie d'apprentissage

### Apprentissage par la pratique

1. **Inspiration** : Voir ce qui existe
2. **Fondations minimales** : Juste assez pour démarrer
3. **Recettes de code** : Morceaux réutilisables
4. **Exploration autonome** : Avec accompagnement
5. **Projet personnel** : Création originale

### Ce que vous apprenez

- Sensibilité au medium : Qu'est-ce que le web peut faire que rien d'autre ne peut
- Pensée systémique : Comment les éléments interagissent
- Design d'expérience : Ce que ressent l'utilisateur
- Critique constructive : Pourquoi ça fonctionne ou pas
- Outils techniques : Comment réaliser vos idées

## L'IA comme assistant

L'IA (ChatGPT, Claude, etc.) est autorisée mais avec responsabilité.

> Si vous ne pouvez pas expliquer le code généré par l'IA, ne l'utilisez pas.

**[→ Guide d'utilisation de l'IA](/rtmf1m/ressources/utilisation-ia)**

## Évaluation

Ce qui compte :

- ✅ Vous comprenez ce que fait le code
- ✅ Le code sert votre intention créative
- ✅ L'expérience est cohérente et engageante

**Répartition :**
- Carnet de recherche/exploration : 20%
- Prototype et itération : 20%
- Projet final : 50%
- Présentation et défense : 10%

**[→ Critères détaillés](/rtmf1m/ressources/evaluation)**

## À retenir

Vous n'allez pas devenir experts Svelte en 32h. Vous allez apprendre à utiliser les outils disponibles pour réaliser vos projets. La syntaxe s'apprend, la vision créative compte plus.
